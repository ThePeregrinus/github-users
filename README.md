## Installation
You need [Node.JS](https://nodejs.org/)
* Clone repo
  * SSH
    * ```git@github.com:ThePeregrinus/github-users.git```
  * HTTPS
    * ```https://github.com/ThePeregrinus/github-users.git```
* Install packages and run
  * ```npm i```
  * ```npm run dev```
